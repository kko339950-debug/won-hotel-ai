# won-hotel-ai
This project is an AI-based assistant designed for the hospitality industry. It helps improve customer service efficiency and communication quality in hotels and service environments.
## Key Features
- Automated responses to customer inquiries (reservations, check-in/check-out guidance)
- Basic multilingual support (including English)
- Suggested responses for various service situations
- Simple customer request management

## How to Use
1. Download or clone the repository
2. Run the program
3. Select and use features based on customer needs
